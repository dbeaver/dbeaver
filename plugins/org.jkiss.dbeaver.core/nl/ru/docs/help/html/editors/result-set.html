Просмотррщик выборки

Выборка - это, как правило, результат выполнения некоторого запроса или содержимое некоторой таблицы или представления.

Просмотррщик выборки. Вы можете копировать любое количество ячеек, выделять столбцы, строки, все ячейки. Выборка может быть отфильтрована и отсортирована в соответствии с заданными пользователем критериями. Содержимое таблицы и результаты выполнения запроса могут быть отредактированы.

Описание

Просмотрщик выборки позволяет просматривать данные в двух режимах: таблично и по записям.

Табличный режим (по-умолчанию) - это типичная таблица, подобная Excel. Режим записи показывает одновременно лишь одну запись в виде двух столбцов - имени колонки базы данных и соответствующего значения. Используйте панель инструментов редактора или клавишу табуляции для переключения между этими режимами.

Для перемещения по столбцам и колонкам используйте стандартные клавиши навигации: стрелки курсора, PgUp, PgDown, Home, End. CTRL+End перемещает к последней строке, CTRL+Home - к первой, и т.д.

Количество показываемых строк ограничено. Максимальное количество строк (значение по-умолчанию - 200) может быть настроено (в меню Параметры или в главной панели инструментов). Если Вы прокручиваете экран до последней показанной строки, и существует возможность загрузить ещё одну порцию данных, DBeaver сделает это автоматически и прокрутит экран дальше.

Копирование и экспорт ячеек

Вы можете выбрать любые ячейки таблицы с помощью мыши в комбинации с клавишами CTRL и SHIFT. Выбранные ячейки могут быть скопированы в буфер обмена в формате, разделённом табуляциями (и затем вставлены в Excel, Calc и т.д.).

Выборка может быть сэкспортирована в файл. Эта команда доступна в контекстном меню. Мастер экспорта поддерживает много форматов экспорта и большое количество параметров. Экспорт будет извлекать все записи из таблицы или результатов запроса (это может занять продолжительное время).

Замечание: DBeaver открывает отдельное соединение с базой данных для экспорта. Это позволяет пользователю продолжить работу с базой данных во время выполнения экспорта.

Фильтрация и сортировка

Выборка может быть упорядочена по одной или нескольким колонкам. Щёлкните по стрелочке рядом с именем колонки для сортировки по ней. Больше настроек сортировки доступно в диалоге фильтрации.

Параметры фильтрации и сортировки могут быть определены в диалоге фильтрации. Он доступен из контекстного меню или при щелчке по левому верхнему углу таблицы.

Ещё информация о диалоге фильтрации.

Замечание: фильтрация/сортировка больших наборов данных может занять продолжительное время, особенно если Вы фильтруете/сортируете по непроиндексированному столбцу. Будьте внимательны при определении критериев фильтрации/сортировки.

Редактирование

Содержимое таблицы и результаты запроса могут быть отредактированы.

Редактирование результатов запроса возможно только если драйвер базы данных поддерживает дополнительные метаданные результатов запроса (сейчас это реализовано и протестировано только для MySQL) и результаты запроса содержат комбинацию столбцов, образующих уникальный ключ.

Редактирование содержимого таблицы возможно, если она имеет по крайней мере один уникальный ключ или индекс.

Диалог редактирования ячейки доступен из контекстного меню. Редактирование прямо в ячейке доступно из панели инструментов (или нажатием CTRL+ALT+S). Для отмены изменений нажмите кнопку "Отмена изменений" (или клавиши CTRL+ALT+R).

Замечание: Если Вы работаете в режиме транзакций (автокомит выключен) тогда Вам необходимо самостоятельно комитить изменения в текущем соединении (команда доступна из главной панели инструментов или главного меню).
