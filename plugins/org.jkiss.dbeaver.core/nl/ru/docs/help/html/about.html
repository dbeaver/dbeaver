Об Универсальном менеджере баз данных

Описание

DBeaver - это свободная универсальная программа для программистов и администраторов баз данных. Её особенности:
  • Удобство и простота использования - главные цели данного проекта, интерфейс программы тщательно спроектирован и реализован.
  • Бесплатна.
  • Работает на множестве платформ.
  • Основана на открытых библиотеках и позволяет писать дополнения (плагины).
  • Поддерживает любые базы данных, для которых существует JDBC драйвер.
  • Может оперировать с внешними источниками данных, для которых может быть, а может и не быть JDBC драйвера.
  • Имеет набор из нескольких плагинов для определённых СУБД (MySQL, Oracle) и различные утилиты для работы с базами данных (реляционные диаграммы).

Поддерживаемые (протестированные) базы данных:

  • MySQL
  • Oracle
  • PostgreSQL
  • IBM DB2
  • Microsoft SQL Server
  • Sybase
  • ODBC
  • Java DB (Derby)
  • Firebird (Interbase)
  • HSQLDB
  • SQLite
  • Mimer
  • H2
  • IBM Informix
  • SAP MAX DB
  • Cache
  • Любой JDBC-совместимый источник данных

Поддерживаемые операционные системы:

  • Windows (2000/XP/2003/Vista/7)
  • Linux
  • Mac OS
  • AIX
  • Solaris

Основные возможности:

  • Просмотр метаданных базы данных
  • Редактор метаданных (таблицы, колонки, ключи, индексы)
  • Исполнение SQL выражений и скриптов
  • Подсветка SQL синтаксиса (специфично для разных СУБД)
  • Автодополнение и гиперссылки на метаданные в SQL-редакторе
  • Редактор содержимого таблиц и выборок
  • Поддержка BLOB/CLOB (просмотр и редактирование)
  • Прокручиваемые результаты запросов
  • Экспорт данных (таблиц, результатов запросов)
  • Управление транзакциями
  • Поиск объектов базы данных (таблицы, колонки, ограничения, процедуры)
  • Реляционные диаграммы
  • Закладки на объекты базы данных
  • Управление SQL скриптами
  • Проекты (соединения, диаграммы, SQL скрипты и закладки)

Возможности MySQL-плагина:

  • Типы данных Enum/Set
  • Просмотр процедур и триггеров
  • Просмотр DDL метаданных
  • Управление сессиями
  • Управление пользователями
  • Управление каталогами
  • Расширенный редактор метаданных

Другие преимущества:

  • DBeaver потребляет значительно меньше оперативной памяти, чем другие подобные программы (SQuirreL, DBVisualizer).
  • Метаданные загружаются по мере необходимости без длительных задержек во время соединения.
  • Просмотрщик выборки (таблица) - очень быстрый и потребляет крайне мало ресурсов.
  • Все операции с базой данных происходят в неблокирующем режиме, поэтому DBeaver не "зависает" в случае проблем с сервером базы данных или других ошибок связи по сети.

Лицензия

DBeaver бесплатен. Вы можете использовать его для любых целей без ограничений. Эта программа предоставляется "как есть" без специальной или подразумеваемой гарантии.

Определённая лицензия на программу и её исходный код пока что не выбраны. В любом случае, её бинарный дистрибутив абсолютно бесплатен.
