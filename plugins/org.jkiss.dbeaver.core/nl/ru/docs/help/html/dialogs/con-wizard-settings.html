Настройки соединения

Общие настройки

В этом месте вы можете определить настройки и параметры соединения.

Некоторые параметры обязательны, другие - опциональны.

Стандартные настройки:

JDBC-адрес Адрес (URL) базы данных. Это поле доступно для редактирования только в том случае, когда драйвер не имеет шаблона адреса (sample URL). В противном случае, это поле будет изменяться автоматически при заполнении прочих параметров соединения.
Хост Имя хоста базы данных
Порт Номер порта базы данных. Как правило, это поле опционально. Если оно оставлено пустым, будет использован порт, принятый по-умолчанию для текущей базы данных.
База данных/схема Имя удалённой базы данных/схемы.
Сервер Имя удалённого сервера.
Имя пользователя/пароль Учётные данные пользователя. Могут быть оставлены пустыми для некоторых драйверов (например, для встроенных баз данных)

Дополнительно

Драйвер базы данных может поддерживать дополнительные настройки соединения. Пользователь может видеть список всех дополнительных свойств на закладке "Дополнительно". Все эти свойства - редактируемые.

Некоторые драйвера поддерживают дополнительные настройки, но не предоставляют их список. В этом случае Вы можете найти список дополнительных свойств в документации драйвера. Выяснив там необходимые параметры, Вы можете определить их в разделе "Пользовательские свойства". Щёлкните там правой кнопкой мыши и выберите пункт меню "Добавить новое свойство".
