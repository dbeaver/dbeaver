Менеджер драйверов

Драйвер базы данных - это сторонняя библиотека, которая позволяет работать с этой базой данных. Большинство драйверов в DBeaver - это JDBC (Java-Database-Connectivity) драйверы, предоставляемые самими разрабочиками баз данных. Диалог менеджера драйверов позволяет Вам создавать, удалять и редактировать описание драйвера.

В диалоге менеджера драйверов Вы можете видеть "дерево" драйверов. Элементы верхнего уровня - "Поставщики драйверов". "Поставщики драйверов" в DBeaver - это плагины. Обычно поставщики драйверов создаются для вполне определённых баз данных, например, для MySQL. Есть одно исключение - обобщённый ("Generic") поставщик, который позволяет Вам работать с произвольной базой данных, имеющей JDBC-драйвер. Обобщённый поставщик ничего не знает об особенностях конкретной базы данных, её специфических функциях, структуре метаданных и т.д. Он лишь предоставляет самый базовый интерфейс.

Вы можете настраивать только "редактируемые" драйверы. Все драйверы из числа обобщённых ("Generic") - редактируемые. Драйвера конкретных баз данных, как правило, не редактируемые, и их параметры не могут быть изменены.

DBeaver предлагается с большим набором предустановленных драйверов. Дополнительно, Вы можете создать свой собственный драйвер для работы в DBeaver.
