Диалог фильтрации и сортировки данных

Вы можете определить критерии для отдельных колонок. Критерии - не просто значения, а части SQL WHERE выражения. Например:

  • = 'Some Value'
  • LIKE '%Value%'
  • BETWEEN 1 AND 1000

Для сортировки по колонке, щёлкните по второй ячейке столбца. Стрелочка показывает направление сортировки (по возрастанию или по убыванию), а номер - порядковый номер столбца в сортировке.
